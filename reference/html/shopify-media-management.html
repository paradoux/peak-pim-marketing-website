FAQ

Everything you need to know about managing your Shopify media with Peak PIM.

Is the media management feature part of Peak PIM or a separate app?

It's built directly into Peak PIM, no separate install or extra cost. Your media library is part of the same workspace where you manage your product data.

What file types does Peak PIM support?

Images (JPG, PNG, WebP, GIF), videos, PDFs, and Shopify 3D models (.glb). Any file type supported by Shopify is supported in Peak PIM.

Can I bulk edit alt text for all my product images at once?

Yes. Peak PIM lets you edit file names and alt text across dozens or hundreds of Shopify images in a single table view. No CSV export, no coding, no one-image-at-a-time editing.

Will my existing Shopify media be imported automatically?

Yes. When you connect your Shopify store, Peak PIM pulls in all your existing files. Your catalog is ready to manage from day one.

Can I manage media across multiple Shopify stores from one place?

Yes. Peak PIM gives you a single shared media library across all your connected Shopify stores. Upload once, reuse everywhere.

How do I find a specific file across a large catalog?

Use folders, tags, and filters to organize your library. Filter by file type, linked product status, or store. No more endless scrolling through Shopify's native files tab.

Can I see which products are using a specific image?

Yes. Every file shows which Shopify products it's linked to. Spot orphaned assets instantly and clean them up with confidence.

Does Peak PIM detect duplicate images?

Yes. Peak PIM surfaces duplicate and near-duplicate files across your Shopify catalog so you can consolidate and keep your library clean.

Can I assign images to products directly from the media library?

Yes. Link images and videos to Shopify products without leaving the media view. No back and forth between screens.

Does better alt text actually improve my Shopify store's SEO?

Significantly. Google uses alt text as a key signal for image search ranking. Merchants with properly optimized alt text see measurable gains in image search traffic. Peak PIM makes it fast to fix at scale.
